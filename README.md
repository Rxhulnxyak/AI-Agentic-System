# ✨ What is Kolimarii?

**Kolimarii** is an advanced **Personal Neural Assistant Platform** engineered to function like a real-world cognitive system.

It is designed as:

🧠 **AI Brain** → Understands, reasons, remembers
🗣 **Voice Interface** → Listens, transcribes, speaks naturally
🖥 **Desktop Agent** → Controls your operating system
📱 **Mobile Bridge** → Connects directly with Android devices
🏠 **Ambient Controller** → Automates smart home devices
🌐 **Web Intelligence Layer** → Searches, reads, analyzes the internet

Kolimarii is not just an assistant — it is a **digital operating companion**.

# 🚀 Quick Start

## Clone

```bash
git clone https://github.com/YOUR_USERNAME/Kolimarii.git
cd Kolimarii
```

## Install

```bash
pip install -r requirements.txt
```

## Configure

Create `.env`

```env
OPENAI_API_KEY=
ANTHROPIC_API_KEY=
MISTRAL_API_KEY=
HASS_URL=
HASS_TOKEN=
ELEVENLABS_API_KEY=
```

## Run

```bash
python main.py
```

Open dashboard:

```text
http://localhost:3000
```
